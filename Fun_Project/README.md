# 🎮 Number Guessing Game using Python

##  Project Description
This is a simple and fun Number Guessing Game built using Python.  
The program generates a random number between 1 and 10, and the user has to guess it within a limited number of attempts.

---

##  Objective
The objective of this project is to understand basic programming concepts such as loops, conditional statements, and user input handling in Python.

---

## ⚙️ How It Works
- The computer generates a random number between 1 and 10  
- The user is given 5 attempts to guess the number  
- After each guess, the program gives hints:
  - "Too high"  
  - "Too low"  
- The game ends when:
  - The user guesses correctly 🎉  
  - OR runs out of attempts 😢  

---

##  Technologies Used
- Python  
- Random Module  

---

##  Features
- Interactive user input  
- Hint system (high/low)  
- Limited attempts  
- Improved version with replay option  

---

## ▶️ How to Run
1. Open the `.ipynb` file in Google Colab or Jupyter Notebook  
2. Run all cells in order  
3. Enter your guesses when prompted
